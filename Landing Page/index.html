Project Name: WorldSurf

Project Description

WorldSurf is going to be the first web 2.0 website for both computers and mobile devices. The site is being designed to provide high-quality detailed information on each surfer destination. Each of these destinations will have content exclusive to members. A big feature for members will be the “My World” section, where members will be able to post info and photos about destinations and create their own destinations (moderated). Becoming a member will be free and require a easy sign-up process. WorldSurf.com will be the only site of its kind with content created for surfers, by surfers.
